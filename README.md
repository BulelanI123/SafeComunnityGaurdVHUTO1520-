# SafeGuard Community

A community safety website that helps neighborhoods report suspicious activity, warn neighbors, and share safety information.

## Project Overview

SafeGuard Community is a web-based platform designed to enhance community safety through:
- Incident reporting system
- Real-time community alerts
- Interactive safety map
- Comprehensive safety tips
- Mobile-responsive design

## Features

- **Report Incidents**: Users can submit detailed reports of suspicious activities, thefts, break-ins, and other safety concerns
- **Community Alerts**: View and stay updated with recent alerts from the community
- **Interactive Map**: Visualize reported incidents on a community map
- **Safety Tips**: Access helpful tips for home security, personal safety, and scam awareness
- **Mobile Friendly**: Fully responsive design that works on all devices

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage for data persistence

## How to Run Locally

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No server required - runs entirely in the browser

## Deployment on GitHub Pages

1. Upload all files to your GitHub repository
2. Go to repository Settings > Pages
3. Select "main" branch as source
4. The site will be available at `https://yourusername.github.io/repository-name/`
